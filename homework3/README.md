# ToDo list

### Description:
This project implements base *ToDo list* functional

It uses Poetry as package manager, sqlalchemy as orm.

Database is maintained in RAM memory by sqlite.
    
### Create venv:
```bash
make venv
```

### Run tests:
```bash
make test
```
    
### Run linters:
```bash
make lint
```
    
### Run formatters:
```bash
make format
```

### Run server:
```bash
make up
```
